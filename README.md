# 🤖 AI Invoice Reconciliation Platform

AI-powered Purchase Order & Vendor Invoice Reconciliation Platform built using Python, Streamlit, PaddleOCR, Groq Llama 3.3 and SQLite.

---

## 🚀 Features

- 📄 Purchase Order & Vendor Invoice Upload
- 🔍 OCR using PaddleOCR
- 🧠 AI-based Information Extraction (Groq Llama 3.3)
- 🔄 Purchase Order vs Invoice Matching
- 💰 Amount & GST Validation
- 🚨 Duplicate Invoice Detection
- 📊 Analytics Dashboard
- 📝 Audit Trail
- 🤖 AI Finance Resolution
- 💳 Payment Approval / Hold Recommendation

---

## 🛠 Tech Stack

- Python
- Streamlit
- PaddleOCR
- Groq API (Llama 3.3)
- SQLite
- OpenCV
- PDF2Image

## ⚙️ Installation

```bash

```
